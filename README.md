# Boids

An installation that treats a language model's embedding space as a proxy for collective
human perception, ranking animals from **beloved to reviled** and letting each group flock
with identical core logic — so the divergence in the viewer's own feeling becomes the exhibit.

This branch (`main`) is the blank baseline: the project brief and repo conventions, no code.
Each development attempt starts from here on its own branch.

- [`PROJECT_BRIEF.md`](PROJECT_BRIEF.md) — authoritative intent: concept, method, and all
  locked decisions. Read it in full before building anything.
- [`DECISIONS.md`](DECISIONS.md) — the running-log convention: one line per non-obvious
  choice, with the reason. Each attempt keeps its own log on its branch.
- [`.env.example`](.env.example) — environment conventions (OpenAI API key for
  curation-time embeddings; the show/runtime needs no API).
